# 🌊 Samudrika | AI Maritime Intelligence

Samudrika is a high-fidelity, military-grade maritime intelligence platform built for real-time underwater image restoration and object identification. This frontend interface features advanced tactical UI components, immersive 3D background simulations, and a velocity-reactive audio engine.

## 🚀 Frontend Development

Follow these steps to set up and run the Samudrika interface locally.

### 1. Prerequisites
Ensure you have **Node.js 18+** installed on your machine.

### 2. Installation
Clone the repository and install the necessary dependencies:

```bash
# Clone the repository
git clone [repository-url]

# Navigate to the project directory
cd samudrika-react-ui

# Install dependencies
npm install
```

### 3. Running the App
Start the development server:

```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## 🛠️ Core Technologies

- **Next.js & React**: Core framework for a high-performance SPA environment.
- **Locomotive Scroll v5**: Precision smooth scrolling with inertia.
- **Web Audio API**: Real-time synthesized "Nuclear Turbine" propulsion sound and tactical click feedback.
- **Three.js & GLSL**: Advanced 3D background simulation (Maritime Teal Hills).
- **Framer Motion**: Smooth tactical UI transitions and automated card cycling.
- **Lucide React**: Clean, high-tech iconography.

## ✨ Key Features

- **Nuclear Turbine Engine**: Rhythmic audio propulsion that reacts dynamically to your scrolling speed.
- **Tactical Analysis Cards**: Automated 3nd-person cycle between Raw, Enhanced, and Metrics views.
- **Maritime Defense Aesthetic**: Deep aquatic theme with glowing cyan topography.
- **Responsive Tactical HUD**: High-contrast, stateful controls designed for precision.

## 🏗️ Project Structure

- `/app`: Next.js App Router pages and global styles.
- `/components/ui`: Custom high-fidelity UI components (Hero, Slider, Header, etc.).
- `/lib`: Shared utilities and constant configurations.
- `/public`: Static assets and imagery.
